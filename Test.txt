Чтобы скомпилировать и запустить программу на Linux, выполните следующие команды в консоли:

php
Copy code
g++ client.cpp -o client -pthread
g++ server.cpp -o server -pthread
./server <port_number>
./client <client_name> <port_number> <period_seconds>
В коде приведены примеры классов TCPClient и TCPServer, которые реализуют клиентскую и серверную части приложения соответственно. Клиент отправляет сообщения с указанным именем через указанный период времени, а сервер принимает сообщения от клиентов и записывает их в файл log.txt.

Клиент и сервер работают в разных потоках, обеспечивая параллельную обработку соединений. Работа с файлом лога происходит в критической секции, чтобы избежать конфликтов при одновременной записи.
